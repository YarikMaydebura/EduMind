# EduMind AI — Student Gamification System

## 🎮 Концепция

Превращаем обучение в RPG-игру где ученики:
- Прокачивают **статы по каждому предмету** отдельно
- Имеют **общий уровень персонажа**
- Видят **профили других учеников**
- Соревнуются за **достижения и leaderboard**
- **Фармят ачивки** выполняя задания

---

## 📊 Система статов

### Структура:

```
ОБЩИЙ ПРОФИЛЬ УЧЕНИКА
├── Overall Level (сумма всех предметов)
├── Total XP
├── Global Achievements
│
├── CLASS: English (with Teacher John)
│   ├── Class Level
│   ├── Class XP
│   ├── Skills: Grammar, Speaking, Listening, Reading, Writing
│   └── Class Achievements
│
├── CLASS: Mathematics (with Teacher Maria)
│   ├── Class Level
│   ├── Class XP
│   ├── Skills: Algebra, Geometry, Problem Solving, Logic
│   └── Class Achievements
│
└── CLASS: Computer Science (with Teacher Alex)
    ├── Class Level
    ├── Class XP
    ├── Skills: Programming, Algorithms, Theory, Projects
    └── Class Achievements
```

## 🗄️ Database Models for Gamification

```sql
-- Student overall stats
CREATE TABLE student_profiles (
    id SERIAL PRIMARY KEY,
    user_id INT REFERENCES users(id),
    total_xp INT DEFAULT 0,
    overall_level INT DEFAULT 1,
    streak_days INT DEFAULT 0,
    last_activity_date DATE,
    avatar_url TEXT,
    profile_frame TEXT,
    badges TEXT[]
);

-- Per-class stats
CREATE TABLE student_class_stats (
    id SERIAL PRIMARY KEY,
    student_id INT REFERENCES students(id),
    class_id INT REFERENCES classes(id),
    class_xp INT DEFAULT 0,
    class_level INT DEFAULT 1,
    skills JSONB DEFAULT '{}',  -- {"grammar": 80, "speaking": 35, ...}
    lessons_completed INT DEFAULT 0,
    homework_completed INT DEFAULT 0,
    quizzes_completed INT DEFAULT 0
);

-- Achievements definitions
CREATE TABLE achievements (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),
    description TEXT,
    icon VARCHAR(50),
    rarity VARCHAR(20),  -- common, uncommon, rare, epic, legendary
    xp_reward INT,
    condition_type VARCHAR(50),  -- streak, level, skill, quiz, etc
    condition_value INT,
    subject VARCHAR(50)  -- null for global, or 'english', 'math', etc
);

-- Student achievements (unlocked)
CREATE TABLE student_achievements (
    id SERIAL PRIMARY KEY,
    student_id INT REFERENCES students(id),
    achievement_id INT REFERENCES achievements(id),
    unlocked_at TIMESTAMP DEFAULT NOW(),
    class_id INT REFERENCES classes(id)  -- null for global achievements
);

-- Student follows
CREATE TABLE student_follows (
    follower_id INT REFERENCES students(id),
    following_id INT REFERENCES students(id),
    created_at TIMESTAMP DEFAULT NOW(),
    PRIMARY KEY (follower_id, following_id)
);

-- Activity feed
CREATE TABLE activity_feed (
    id SERIAL PRIMARY KEY,
    student_id INT REFERENCES students(id),
    activity_type VARCHAR(50),  -- achievement, level_up, quiz_complete, etc
    description TEXT,
    metadata JSONB,
    created_at TIMESTAMP DEFAULT NOW()
);
```

## 🎯 Achievement Categories

### Global Achievements
- Streak achievements (3, 7, 14, 30, 100 days)
- Level achievements (5, 10, 25, 50, 100)
- Homework achievements (first perfect, 10 perfect, never miss)
- Social achievements (first follow, 10 friends)
- Consistency achievements (perfect week, perfect month)

### Per-Subject Achievements (English, Math, CS)
- Skill master (80%+ in specific skill)
- Subject polyglot (all skills 80%+)
- Quiz champion (10 quizzes 100%)
- Course completion
- Speed learner (complete X lessons in Y time)

### Special/Hidden Achievements
- Night owl (study after midnight)
- Early bird (study before 6am)
- Marathon (5 hours in one day)
- Comeback (return after 30+ days)
- Helper (assist classmates)
